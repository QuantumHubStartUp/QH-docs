

# QH-Frontend-web

## Описание

Данные проект является, frontend частью проекта
QH (Quantum Hub). Web версия приложения.

---

## Запуск в development:

* Linux:
```bash
./cmd/linux/dev.sh
```

* Windows:
```bash
./cmd/windows/dev.bat
```


## Запуск в pre-production:
1. Установить docker и docker-compose
2. ```bash
    ./cmd/linux/pre-prod.sh
   ```
