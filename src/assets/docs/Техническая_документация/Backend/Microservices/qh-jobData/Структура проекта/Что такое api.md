# Что такое api

API - это интерфейс для взаимодействия между различными компонентами системы.

## Структура api

```bash
├── app
│   ├── api
│   │   ├── __init__.py
│   │   ├── routes.py
│   │   └── v1
│   │       ...
│   │
```

## Структура v1

```bash
├── app
│   ├── api
│   │   └── v1
│   │       ├── __init__.py
│   │       ├── user_router.py

```

v1 - это папка для версий api.
